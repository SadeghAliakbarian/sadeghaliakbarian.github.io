Consider we have a model $p_\theta(x, z)$, that works with a set of observed variables $$x$$, which we have access to given a dataset, and a set of unobserved \emph{latent} variables $$z$$, which typically are not available in the dataset. This model $$p_\theta(x, z)$$ aims to model the observations, i.e., $$x$$ through a neural network parameterized with $$\theta$$. The likelihood of the data can then be computed as the marginal distribution over the observed variables.
